### How to run?

1. clone this repository
```
git clone --recursive [URL_HERE]
```
2. install required packages from `requirements.txt`
3. run FastAPI app using terminal command: 
```
uvicorn src.main:app --reload
```
4. go to: http://127.0.0.1:8000/
